Astromech Droid Controller

This project involved designing and building the main controller board for an Astromech droid, integrating multiple subsystems such as power management, motor control, and telemetry.

Technologies Used

  • ESP32-S3 Microcontroller
  • PCB design with KiCad
  • I2C, UART, and PWM interfaces
  • Custom firmware in C/C++

Challenges & Solutions

One of the main challenges was ensuring fault tolerance for motor drivers; this was solved by implementing current sensing and auto-shutdown protocols.
